What does it mean to display inline vs inline blocks?

INLINE - making things...inline, side by side, next to each other. This function helps us display items side by side. If you are familiar with Microsoft word it is basically similar to the text wrapping function.

We use the css property display to call out this different values i,e

  • Display: inline;
  • Display: inline-block;

Display: Inline

Puts two elements side-by-side. Height and width of the element is not adjustable.

inline

inline

Display: Inline-block

Puts two elements side-by-side but you will be able to adjust the height and width
