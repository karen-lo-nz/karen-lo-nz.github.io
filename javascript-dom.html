Fundamentals of Javascript

HTML vs CSS

HTML provides structure and CSS provides the style. I.e architect designs the build of a building, while the interior designer decides on the style and scheme for the inside of the building to fill the inside with beauty.

Control Flow and Loops

Control Flow notes the order of statements/code that is being run in a document. In coding, it is read from top to bottom. This is always the case unless there is another set of code to say otherwise. This is where the loop comes in. Instead of continue reading down, the loop can instruct the code to change direction/repeat.

For example - You want to water all the plants in your house. Instead of saying water plant a, plant b and plant c, you could say:

While unwatered plants > 0, keep watering plants

DOM - Document Object Model

DOM refers to everything inside the page. It provides a visualisation of the directory of the page. I.e like a family tree or an organisation chart.
Similar to using CSS to select and modify elements in html for styling purposes, we use JavaScript to select and modify elements in a html document using tag, class or id to perform something. To access any elements in a page, you use the DOM (like a directory) to find it by tags (p, h1, h2), classes, or ID

To find these element, use the Javascript function of querySelector( ) or querySelectorAll( )

Example 1: document.querySelector(".blue")

This will return all the first element in the page using class blue.

Example 2: document.querySelectorAll(".blue")

This will return all the elements in the page with class blue.

Accessing arrays and objects

Both arrays and objects are a collection or group. There are different ways to access them:

Arrays -access by using an index list and is arranged in a sequence with numbered ID.

let myFruit = [“apple”, “red”, “autumn”]

In this Array, the index list will be 0= apple, red = 1 and autumn = 2 *Note index always starts from 0

Objects - access by using the dot notation or bracket.

let myFruit = {

name: “apple”,

colour: “red”,

season: “autumn”

}

In this Object, to access any of the values we use the dot or bracket notation. I.e myFruit.name , myFruit.colour, myFruit.season or myFruit[name],myFruit[colour], myFruit[season]

Using the array and object above, we can return “apple” when

for array: console.log(myFruit[0]);

for object: console.log(myFruit.name);

Functions

Functions in Javascript allow for a specific set of code to be reused with any parameters. This is useful as rather than writing out the same code each time we need it, we simply call the function as many times we like to perform the same action.

For example, if we wanted the code to do an addition of three inputs/parameters, we can create a function

function add( a, b, c){ // the bracket is where you input the parameters

Return (a + b + c);

}

Anytime we need to add 3 items, we can pass it into the add function and it will return its total i.e

add(1,2,3);

This will return 6
